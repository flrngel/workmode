How to add blocked sites — one entry per line:
  • Plain text — blocks any URL that contains the text (case-insensitive).
    Example: reddit.com/ blocks all of Reddit.
  • Regex /pattern/ — blocks URLs matching the regular expression.
    Example: /^https?:\/\/(www\.)?x\.com\// blocks only x.com.
    ⚠ Plain text like x.com also matches beanspdx.com because "x.com" appears inside it. Use a regex for exact-domain matching.
A blank line is ignored.
